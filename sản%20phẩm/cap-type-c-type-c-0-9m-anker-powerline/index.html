Mô tả

Đầu ra 20V – 3A, đáp ứng nhiều mức sạc khác nhau, cáp sạc Anker có thể sạc Samsung S10 lên 50% chỉ trong 30 phút khi kết hợp với adapter Power Delivery 18W.

Tích hợp đầu cáp Type C – Type C tương thích tốt với nhiều dòng máy

Anker PowerLine III A8852 dùng sạc, trao đổi dữ liệu hiệu quả, tương thích 100% với các điện thoại, laptop, tablet: iPad Pro 2018, Galaxy Note 8, S10/S10+, Google Nexus 6P, Huawei Matebook, MacBook Pro 2019/MacBook Air 2019 và nhiều dòng thiết bị điện tử khác.

Tương thích nhưng có sự hạn chế về tốc độ sạc gồm dòng Google Nexus 5X, MacBook Pro 2016, Lumia 950, LG V20/G5/G6, Xiaomi 5, Huawei P30/P30 Pro.

Thiết kế gọn nhẹ, có thể chịu đến hơn 25.000 lần uốn cong

Cáp sạc có dây bền dẻo, dễ cuộn gọn khi cần, màu đen sang trọng, duy trì độ mới lâu. Dây cáp dài 0.9 m phối ghép các thiết bị dễ dàng, vừa đủ độ dài để kết nối điện thoại với máy tính để bàn thoải mái, không sợ thiếu hụt dây.

Sao chép dữ liệu nhanh chóng giữa các thiết bị

Khi cáp Type C – Type C có thể truyền phim ảnh, danh sách nhạc hoặc toàn bộ thư viện ảnh chỉ trong vài giây với tốc độ lên tới 480 Mbps.

Nhìn chung, với công suất ấn tượng, độ tương thích cao, thiết kế bền chắc nhỏ gọn và chức năng sạc, truyền dữ liệu tốt, cáp Type C – Type C Anker PowerLine III A8852 hứa hẹn sẽ mang đến nhiều tiện ích cho cuộc sống của bạn
